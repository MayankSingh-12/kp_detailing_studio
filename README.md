# KP Detailing Studio

Live site: https://mayanksingh-12.github.io/kp-detailing-studio

Faridabad's Premier Auto Detailing Studio — premium dark-luxury website built with pure HTML, CSS, and vanilla JavaScript.

## To add videos

Upload `.mp4` files to the `/videos/` folder:

- `videos/video1.mp4`
- `videos/video2.mp4`
- `videos/video3.mp4`

The video player will automatically display them in the **See The Transformation** section.

## Structure

```
kp-detailing-studio/
├── index.html        — main page
├── style.css         — all styles
├── js/
│   └── main.js       — scroll animations, hamburger, fade-ins
├── videos/
│   └── .gitkeep      — placeholder (add .mp4 files here)
└── README.md
```

## Tech

- Pure HTML5, CSS3, Vanilla JS — no frameworks, no dependencies
- Google Fonts: Bebas Neue + Rajdhani
- Fully mobile responsive
- Scroll fade-in animations via IntersectionObserver
